# 欢迎来到 Naikangyuan 的文档库

> **以有限元方法为舟，以高性能并行计算为海。**

## 🎯 网站宗旨

本站致力于构建一个连接 **地球物理数值模拟** 与 **高性能并行计算 (HPC)** 的知识体系。我的核心目标是通过具体的工程实例，打通从“软件应用”到“算法设计”的技术路径。

## 🗺️ 内容路线图

在这里，你将看到以下三个维度的内容深度：

1.  **应用层：ASPECT 与地球物理软件**
    *   ASPECT 软件的基础使用、参数调试与案例复现。
    *   其他地学领域常用软件的横向评测与应用技巧。

2.  **底层：deal.ii 与有限元开发**
    *   深入剖析 ASPECT 强大的底层引擎—— **deal.ii** 有限元代码库。
    *   分享我在源码阅读、功能扩展以及二次开发过程中的实战经验。

3.  **核心目标：并行计算与数值算法**
    *   这是本站最终通向的“宽阔大道”。
    *   通过上述具体的软件实例，提炼通用的**有限元方法 (FEM)** 逻辑与**大规模并行计算 (MPI/OpenMP)** 策略，探索计算科学的无限可能。
